# Public API Playground

This project is a simple web application that combines multiple public APIs into one interface. It demonstrates how to fetch and display data using JavaScript.

## Features

1. Dog Finder
- Fetches a random dog image
- Displays the breed name from the image URL
- Copy image URL button

2. Joke Generator
- Shows a random joke (setup and punchline)
- Next joke button

3. Random User Profile
- Displays user name, profile picture, email, and country
- Also shows age or phone number

4. JSONPlaceholder Posts
- Fetches and displays a list of post titles

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Open index.html in your browser

## APIs Used

- https://dog.ceo/api/breeds/image/random
- https://official-joke-api.appspot.com/random_joke
- https://randomuser.me/api/
- https://jsonplaceholder.typicode.com/
